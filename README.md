# vue-docker-azure

Dockerizable Vue 3 template with Pinia associated with Axios

### Technologies Used

1. Vue3, Vite, Pinia, Axios, Devextreme
2. Typescript
3. Node, Express
4. Jest
5. Docker
6. Socket
7. Nginx

### Contributors

Hyunbae Jeon 100%

### Running Front

RUN

1. cd front
2. yarn install
3. npm run dev

Test

1.  npm run test:unit

### Running Server

RUN

1. cd server
2. npm i
3. npm run dev

### Build Docker Image

1. docker-compose-front up --build
